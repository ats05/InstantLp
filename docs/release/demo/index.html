この記事はGoogleDrive上に設置した.docx形式のファイルから生成しています。

挿入した画像はそのまま表示ができます。

が、サイズや位置の指定をするためには開発が必要です。

フォーマットについて

wordの見出し1、見出し2、見出し3を設定すると、記事の内容にも反映されます。

  • 箇条書きの表も
  • 入力した通りに
  • 表示されます。
  1. 番号が付いていても
  2. 同様です。

ハイパーリンクを設定することも可能です。

ファイル名について

ファイル名の頭に「【公開】」をつけると、公開されます。

「release/ファイル名」が記事のURLになります。
